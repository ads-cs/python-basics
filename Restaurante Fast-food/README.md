# Restaurante Fast-Food em Python

Projeto em Python que simula um sistema simples de pedidos em um restaurante fast-food,
com cardápio, carrinho de compras e descontos progressivos.

## Funcionalidades
- Escolha de categorias
- Adição de itens ao carrinho
- Cálculo automático de descontos
- Interface em terminal

## Tecnologias
- Python 3

## Objetivo
Projeto criado para praticar lógica de programação, listas, funções e estruturas de controle.
